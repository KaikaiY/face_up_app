# Face Up

## アプリ概要
Face Upは、数学に苦手意識を持つ子どもが、問題を解いた後に「なぜできたのか」「どこでつまずいたのか」を振り返るための学習支援アプリです。

一般的な学習アプリのように問題数や正答率だけを重視するのではなく、正解・不正解の理由を記録することで、学習者自身が自分の理解度やつまずきの原因に気づけることを目的としています。

## 制作背景
私には、勉強に苦手意識を持つ妹がいます。
現在の日本では、5教科の成績が進路選択に大きく関わる場面が多くあります。その中で、妹には「自分は勉強が苦手だ」と感じ続けてほしくないと考えました。

妹は、嫌いな勉強であっても宿題をそつなくこなし、やるべきことを早く終わらせる力があります。
そのため、能力がないのではなく、勉強に対する苦手意識や、自分がどこでつまずいているのかが見えにくいことが課題なのではないかと考えました。

そこで、問題を大量に解くことよりも、問題を解いた後の感想や気づきを記録し、「なぜできたのか」「なぜ間違えたのか」「偶然できたのか」「理解してできたのか」を振り返るアプリを作成しました。

## 解決したい課題
- 勉強に対して「自分は苦手だ」と思い込んでしまう
- 正解・不正解だけで理解度を判断してしまう
- どこでつまずいているのかが本人にも周囲にもわかりにくい
- ケアレスミス・理解不足・偶然の正解が区別されにくい
- 学習の成功体験が記録として残りにくい

## このアプリでできること
- 数学の問題を解く
- 解答後に、正解・不正解だけでなく理由を振り返る
- 「説明できる」「なんとなく当たった」「計算ミスをした」などの振り返りを記録する
- 自分のつまずきや得意な解き方を見える化する
- 学習者が「全部苦手」ではなく「ここを練習すればよい」と気づけるようにする

## こだわったポイント

### 正誤よりも振り返りを重視

このアプリでは、問題の正答率だけで学習者を評価しません。
正解した場合でも「理解して正解したのか」「なんとなく当たったのか」を記録できるようにしています。
また、不正解の場合も「計算ミス」「問題文の読み違い」「解き方がわからなかった」など、原因を分けて記録できます。

### 苦手意識を強めない表現

学習者に対して「苦手」「できない」と決めつけるのではなく、
「ここを練習するともっと楽になりそう」という前向きな表現を意識しています。

### 家族が支援しやすい設計

学習者本人だけでなく、家族や支援者が「どのように声をかければよいか」を考えやすいように、学習結果や振り返り内容を記録できる設計を目指しています。
